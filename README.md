# 操作方法 (multiple view modes)

## 視点・カメラ操作
- **マウス移動**: 視点移動（カメラの回転）
- **Vキー**: 視点の切り替え（一人称視点 / 三人称視点）

## 移動
- **Wキー / ↑キー**: 前進
- **Sキー / ↓キー**: 後退
- **Aキー / ←キー**: 左移動
- **Dキー / →キー**: 右移動

### 一人称視点モード (地上移動)
- 方向キーやWASDキーで、カメラの向いている方向（水平方向）へ移動

### 三人称視点モード (空間移動)
- カメラの向いている全方向へ自由に移動
- **Eキー**: 上昇
- **Qキー**: 下降
